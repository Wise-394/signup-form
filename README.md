# 📝 Sign Up Form Project (The Odin Project)

This is a sign-up form project built as part of __The Odin Project__ curriculum. It focuses on HTML form structure, CSS styling, and JavaScript client-side form validation to create a realistic and user-friendly registration experience.

## 🚀 Features

* Real-time input validation with visual feedback
* Custom error messages for invalid or missing fields
* Password strength and match checking
* Responsive layout

## ✅ Validation Rules

* **Required fields** — All fields must be filled before submission
* **Email format** — Must follow a valid `user@example.com` pattern
* **Password** — Minimum length enforced with character requirements
* **Password confirmation** — Must exactly match the password field

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)
